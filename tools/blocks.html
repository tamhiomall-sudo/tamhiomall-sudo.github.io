New block

Type a name that is not in the list to create a new category or element. Element is optional: leave it blank where blocks sit directly under the category.
Markdown subset: # ## ### headings, - bullets, | tables |, **bold**. Placeholders look like {{threat_vehicle}}.

Library

Import blocks

Paste one or more blocks below and import them in one pass. Everything imported lands as draft: you mark a block ready yourself, once you have read it. Import never overwrites an existing block; it adds new ones.

Format: a few Key: value header lines (Category, Element, Order, Title), then a line with three dashes ---, then the block body. Separate one block from the next with a line of three equals signs ===. Element and Order are optional. Status is ignored on import: everything comes in as draft.
